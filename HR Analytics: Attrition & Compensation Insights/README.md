# HR Analytics: Attrition & Compensation Insights

## Objective
Assess attrition patterns and compensation structure to support workforce stability.

## Dataset
1,480 employees | 34 HR features

## Key Analysis
- Attrition vs Age / Tenure / Income
- Correlation analysis
- Compensation fairness evaluation

## Key Findings
- Early-career attrition concentration
- Short tenure associated with higher turnover
- Income and job level linked to exit exposure
- No significant gender pay disparity observed
